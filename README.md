# Password Manager Android

## Project Overview

Password Manager Android is a secure and user-friendly application designed to help users manage their passwords efficiently. With this app, users can store, retrieve, and manage their passwords securely in one place.

## Features

- **Secure Storage**: All passwords are encrypted using AES encryption.
- **Add Passwords**: Easily add new password entries.
- **View Passwords**: View stored passwords in a user-friendly interface.
- **Modify Passwords**: Update existing password entries.
- **Delete Passwords**: Remove passwords that are no longer needed.
- **Search Functionality**: Quickly find passwords using the search feature.
- **User-Friendly Interface**: Intuitive design for seamless navigation.

## Steps to Use

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/anirruth0312/password_manager_android.git
   ```
2. **Open the Project**:
   - Open the project in Android Studio.
   - Ensure that you have the latest version of Android Studio installed.

3. **Set Up the Environment**:
   - Add a `local.properties` file in the root directory with the following content:
     ```
     sdk.dir=<path-to-your-android-sdk>
     ```

4. **Build the Project**:
   - Sync the Gradle files.
   - Build the project to ensure all dependencies are resolved.

5. **Run the Application**:
   - Connect an Android device or use an emulator.
   - Run the app from Android Studio.

6. **Use the App**:
   - Navigate through the app to add, view, modify, or delete password entries.
